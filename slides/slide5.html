Network Security - Core Concepts

Network Fundamentals

  • A network connects multiple devices for communication.
  • Protocols define rules for communication between devices.

OSI Model (7 Layers)

  • Application: Interface for user services (HTTP, FTP).
  • Presentation: Data formatting, encryption (SSL/TLS).
  • Session: Session management (RPC).
  • Transport: Reliable data transfer (TCP, UDP).
  • Network: Routing and addressing (IP, ICMP).
  • Data Link: Framing data for transmission (Ethernet, ARP).
  • Physical: Physical transmission of bits (Cables, Wi-Fi).

TCP/IP Model (Simplified)

  • Application Layer (HTTP, DNS)
  • Transport Layer (TCP, UDP)
  • Internet Layer (IP, ICMP)
  • Link Layer (Ethernet, ARP)

Firewalls

  • Packet Filtering Firewall: Filters traffic by IP, port, protocol.
  • Stateful Inspection Firewall: Tracks active connections.
  • Proxy Firewall: Inspects packets at application layer.

Common Network Threats

  • Interruption: Denial of service (DoS) attacks.
  • Interception: Sniffing, Man-in-the-Middle attacks.
  • Modification: Altering data during transmission.
  • Fabrication: Injecting false messages into the network.

Network Security Tools

  • SSL/TLS: Protects data in transit (HTTPS).
  • IPSec: Protects IP packets with encryption/authentication.
  • VPNs: Secure tunnels over public networks.
  • IDS/IPS: Intrusion Detection and Prevention Systems.
